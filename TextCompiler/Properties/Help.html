Working with Files

Creating a New File

In the File tab, select Create (Ctrl+N)

Opening a File

In the File tab, select Open (Ctrl+O) or drag and drop a file into the editing window

Saving a File

In the File tab, select Save (Ctrl+S) or Save As to save the file under a different name

Closing a File

In the File tab, select Exit (Ctrl+W) to close the current file

Working with Text (Edit)

Undo/Redo - undoes the last action or redoes the undone action

Cut - cuts the selected text and copies it to the clipboard

Copy - copies the selected text to the clipboard

Paste - pastes text from the clipboard into the editing window

Delete - deletes the selected text

Select All - selects all text in the editing window
